## File Formats

Each model folder includes:
- **Native CATIA files** (`.CATPart`, `.CATProduct`) — full part design and assembly
- **STEP file** (`.stp`) — for viewing in free CAD tools without a CATIA license
- **Rendered screenshot** (`.png`/`.jpg`) — quick visual reference

## Software

CATIA V5 — Part Design, Generative Shape Design (GSD), Assembly Design
